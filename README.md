# 🌍 IoT-Based Air Quality & Pollution Monitoring Dashboard

## 📌 Overview

An IoT-inspired environmental monitoring system that simulates real-time air quality, temperature, and humidity data and visualizes it through an interactive Streamlit dashboard.

The system continuously generates sensor readings, stores them in CSV format, performs AQI classification, and displays analytics and alerts for environmental monitoring.

## 🚀 Features

* Real-Time Air Quality Monitoring
* AQI Classification (Good, Moderate, Poor, Hazardous)
* Temperature & Humidity Tracking
* Interactive Analytics Dashboard
* Automated CSV Data Logging
* Alert Generation for High Pollution Levels
* Streamlit-Based Responsive UI

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* CSV Data Storage

## ▶️ Run Project

1. Install dependencies

pip install -r requirements.txt

2. Start simulator

python simulator.py

3. Launch dashboard

streamlit run Home.py
